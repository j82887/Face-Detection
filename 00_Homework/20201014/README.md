# 02. 貓狗影像辨識
#### 1. 使用彩色多輸入的影像
#### 2. 模型訓練以Kears來自行設計，其模型需以卷積網路來建立
    a. 需呈現訓練過程的學習曲線（包含訓練集與驗證集，其中x軸為迭代次數，y軸為準確度與誤差）
    b. 需呈現訓練與測試集的預測結果，混淆矩陣與準確度
#### 3. 設計GUI介面：
    a. 視窗名稱與標題
    b. 插圖與預測標籤文字
    c. 按鈕觸發事件，事件為選擇影像與更換影像，並進行影像帶入模型之預測，其預測結果來更換預測標籤文字
